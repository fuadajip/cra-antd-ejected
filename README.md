# SavingNextGen CMS

CMS for counseling or education for pregnant and lactating mothers about health aspects during pregnancy, child health, and other things well.

## Installation

Make sure you have nodejs v.8*+.
[Install Nodejs](https://www.nodejs.org)

#### Install yarn using npm<br/>
```bash
npm i -g yarn
```

#### Install Packages<br/>
```bash
yarn
```

#### Run Webpack<br />
```bash
yarn start
```

#### Run unit Test<br />
```bash
yarn test
```

#### Run Prettier<br />
```bash
yarn prettier 
```

#### Watch Prettier<br />
```bash
yarn prettier-watch
```

#### Watch Flow<br />
```bash
yarn flow:watch
```

#### Production Build <br/>
```bash
yarn build
```
